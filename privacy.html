Dasar Privasi untuk KakifilemBot

1. Data yang Dikumpulkan

KakifilemBot hanya mengumpulkan User ID pengguna Telegram apabila pengguna berinteraksi dengan bot.

2. Tujuan Penggunaan Data

  • Mengenal pasti pengguna untuk membalas arahan bot.
  • Mengesan log aktiviti untuk tujuan pembaikan atau penyelenggaraan jika berlaku masalah pada bot.
  • Mengenal pasti pengguna premium berdasarkan User ID.

3. Penyimpanan Data

User ID disimpan hanya untuk tempoh yang diperlukan bagi tujuan perkhidmatan bot. Data ini tidak akan dikongsi dengan pihak ketiga tanpa kebenaran pengguna.

4. Keselamatan Data

Kami mengambil langkah-langkah munasabah untuk melindungi data anda dan memastikan ia dikendalikan dengan selamat.

5. Tiada Data Peribadi Sensitif

Kami tidak mengumpul maklumat peribadi sensitif seperti nama, alamat emel, atau lokasi melainkan dinyatakan secara jelas.

6. Persetujuan Pengguna

Dengan menggunakan KakifilemBot, anda bersetuju dengan pengumpulan dan penggunaan data seperti yang dinyatakan dalam dasar privasi ini.

7. Cara Menghubungi Kami

Jika anda mempunyai sebarang soalan berkaitan dasar privasi ini, sila hubungi kami di @kakifilemreport1.
